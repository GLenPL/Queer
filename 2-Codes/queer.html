Quelques explications

Tous les sripts de cette page sont codés en javascript et accesible ici. Un code dessine toute les 40 millisecondes sur l'élément central de la page page, donnant une illusion de mouvment. Losrque le bouton ad hoc est utilisé, toutes les frames sont enregistrées une par une (le processus est relativement long et peu facilement prendre plus de 40ms, ce qui explique le ralentissement de l'animation) avant d'être utilisées pour créer un gif. Celui-ci est alors téléchargé. Tout est fait localement dans le navigateur.

Pour les explications à propos des paramètres, il va falloir attendre un joli schéma (et donc beaucoup de motivation de ma part)
